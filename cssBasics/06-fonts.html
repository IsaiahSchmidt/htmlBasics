font

in css, there are two types of font family names

  • generic familya group of font families with a similar look (ex: serif)
  • font familya specific font family (ex: times new romas, arial, etc)

the font family property should hold several font names as a fallback in case the browser doesnt know/support the font you first picked

always start with the font you want, and end with a generic family

font families that are more than one word must be held inside "quotation marks" (ex: "times new roman")

css font properties

font
sets all of the font properties in one declaration
font family
specifies the font family for the text
font size
specifies the font size of the text
font weight
specifies the font weight
