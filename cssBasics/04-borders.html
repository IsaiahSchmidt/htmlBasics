borders

the border property allows you to specify the style, width and color of an elements border

  • dotted:dotted border
  • dashed:dashed border
  • solidsolid border
  • doubledouble border
  • groove3d grooved border, depends on border-color value
  • ridge:ed ridged border, depends on border-color value
  • inset3d inset border, depends on border-color value
  • none:no border
  • hiddenhidden border
  • mixedyou can set a different border style for the top, bottom, left, right

border width

very important: border-width will not work unless you have 'border-style' set

border-width can be set to a specific size (px, em, etc) or you can use a preset setting: thin, med, thick. the border-width property can also take up four values: top, right, bottom, left

border color

very important: border-color will not work unless you have 'border-style' set

border-color can be set in the following ways:

  • name:specify color, such as blue
  • hex:specifies hex value such as '#eee'
  • rgb:specifies rgb value, like 'rgb(100,100,100)'
  • transparent:this is the 'alpha' channel
