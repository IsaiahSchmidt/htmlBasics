width and height notes

the width and height properties are used to set the width and height of an element

the default setting is auto, which means the browser decides the width and height

other settings for width and height are:

  • specified length values such as px and em
  • percentage of the containing element (ex:50%, 100%, etc)

this has a width of 50% and a height of 200px

you can also manipulate width and height with images
