Font

in CSS, there are two types of font family names:

  • generic family: a group of font families with a similar look (ex: serif)
  • font family: a specific font family (ex: times new roman or ariel)

the font-family property should hold several font names as a fallback in case the browser does not support the first family you picked

always start with the font you want, and end with a generic family

font families that are more than one word must be held inside quotation marks (ex: "times new roman")

CSS font properties

font
sets all the font properties in one declaration
font-family
specifies the font family for the text
font-size
specifies the font size for the text
font-weight
specifies the font weight of a font
