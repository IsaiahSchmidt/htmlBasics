CSS IDs

an ID is a unique identifier

ID notes

  • in css an ID starts with an octothorpe (pound, hastag, number, etc)
  • example uses for IDs: targeting a specific header or an about section
  • each id should only be used once per page - they are unique
    this is important with javascript later on as well
  • use for one off styling

IDs vs Classes

IDs

this ID will be linked in the css page by a #

you can't have two IDs on an element. if you do, neither will be applied

Classes

elements can have any number of classes

for example, this element has two classes

you CAN use an ID twice, but you shouldn't
