Borders

the border property allows you to specify the style, width and color of an element's border

border style

the border-style property can take up to four values (top, right, bottom, left) the following border styles are allowed:

  • dotted: dotted border
  • dashed: dashed border
  • solid: solid border
  • double: double border
  • groove: 3d grooved border, depends on border-color value
  • ridge: 3d ridged border, depends on border-color value
  • inset: 3d inset border, depends on border-color value
  • none: no border
  • hidden: hidden border
  • mixed: you can set a different border style for the top, right, left, and bottom

border width

Very Important: border-width will not work unless you have border-style set

border-width can be set to a specific size (px, em, etc) or you can use one of the following pre-set settings: thin, medium, thick. the border-width property can also take up 4 values (top, right, bottom, left)
