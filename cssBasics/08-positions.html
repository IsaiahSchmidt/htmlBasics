position

the position property specifies the type of positioning method used for an element

the four position values are:

  • fixedthis will pin something to a spot and it will stay there as the user scrolls (ex: nav bars)
  • staticthis is the default setting. not affected by top, bottom, right, left properties
  • relativethis doesnt take the element out of normal document flow. still appears w/n its space/div/container, but allows it to be offset a bit from top, bottom, left, right. these properties will move it away from its normal position. other content will not be adjusted to fill any gap left by the element
  • absolutetakes an element out of document flow. if a parent element is set to relative, the child is absolute, the child elements new (0,0) cordinate will be the upper left corner

    • the z-index allows you to override the stacking of elements in a page. the default value is 0. you can set this value either positive or negative. positive=front, negative=behind.
