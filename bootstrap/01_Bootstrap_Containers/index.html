The Bootstrap Containers

Default Container

fixed-width container, means max-width changes at each break point.

Responsive Containers => specify a class is 100% wide until the specified breakpoint is reached

100% wide until small breakpoint
100% wide until med breakpoint
100% wide until lg breakpoint

Fluid Containers => full width container spanning the whole width of the viewport
