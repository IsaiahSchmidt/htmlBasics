Default Grid Setup

  1. you can control the grid by the 'six-responsive breakpoints'
  2. containers center and horizontally pad your content
  3. rows are wrappers for columns => each column has horizontal padding( called a gutter) for controlling the space in between them. these gutters can be modifie via bootstrap modifiers
  4. columns are felxible => there are 12 template columns available per row, so you can create many different combinations of elements that span the number of columns. columns classes indicate the number of template columns to span (ex. col-4, this column takes up a span of 4). widths are set in %s, so you'll always have the same relative sizing
  5. gutters are also responsive and customizable => gutter classes are available across all breakpoints, with all the same sizes as our 'margin and padding spacing'. if you want to change horizontal gutters, use: .gx-* classes, use vertical gutters: gy-* classes, and you can use g-0 to remove gutters.
